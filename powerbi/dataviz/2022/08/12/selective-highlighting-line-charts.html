Conditionally Highlighting Data Labels

Conditional formatting is not new in Power BI however in August 2022 release a new option was added for conditionally highlighting data labels in charts. It works the exact same way by clicking on the "fx" button in formatting options. This opens new way of selectively highlighting data labels by changing colors and color transperancy. I have written about changing color transperancy using HSLA in my previous blog so I will directly show the the DAX and the before/after.

In the chart below, I have added conditional formatting to only show values that are above 10%. You can use any complex logic that can be written in DAX.

DAX With HSLA

To pass color transperancy to the color format, use the alpha value in the HSLA. 0 means transperant and 1 is opaque. Values between 0 and 1 will show various degree of transperancy.

Add the DAX measure to the "fx" as shown in the video below to get the desired effect.
